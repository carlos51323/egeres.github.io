Understanding the blockchain...

A blockchain itself is defined as :

" A continuously growing list of records, called blocks, which are linked and secured using cryptography. "

Okay... uhm... so what ?

First... now days you might have heard of all theese new crypto currencies like bitcoin, ethereum or many others.

Well, they are a way to storage money as digital information, but, we are talking about money, how do you make 100% sure nothing of this system is vulnerable to hackers ?

Here is where the blockchain is needed, a de-centralized way to storage information such as no arbitrary changes can be made and all the transactions are encrypted

You can think of the blocks of a blockchain as something like this :

A set of information that contains all the transaction hashes combined with the previous block hash
